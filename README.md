# Web-Beest - SEO
This recipe is designed to configure a list of modules and configuration needed
for SEO in a Drupal site.

## Features
Adds and configures the following modules:
* [Favicons](https://drupal.org/project/favicons)
* [GDPR](https://drupal.org/project/gdpr)
* [HTTPSWWW redirect](https://drupal.org/project/httpswww)
* [Metatag](https://drupal.org/project/metatag)
* [Pathauto](https://drupal.org/project/pathauto)
* [Redirect](https://drupal.org/project/redirect)
* [RobotsTxt](https://drupal.org/project/robotstxt)
* [Schema Metatag](https://drupal.org/project/schema_metatag)
* [Yoast SEO](https://drupal.org/project/yoast_seo)
* [XMLSitemap](https://drupal.org/project/xmlsitemap)
